RestFrames::MinMassDiffInvJigsaw Class Reference

Detailed Description

Minimizes the absolute different of masses of N_vis_inv_pair pairs (see constructor below), usually used with 2 children however when more than 2 children minimizes the sum of mass differences.


Read more about the MinMassDiffInvJigsaw on the "Recursive Jigsaw Reconstruction report". Jigsaw Rule VII.3 (Invisible Minimize âˆ†Masses).
