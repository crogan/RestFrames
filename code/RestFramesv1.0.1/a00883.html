RestFrames::MinMassDiffCombJigsaw Class Reference

Detailed Description

Minimizes the absolute difference between the masses of N_mass (see constructor below) objects, usually used with 2 children however when more than 2 children minimizes the sum of mass differences.


Read more about the Combinatoric Jigsaws on the "Recursive Jigsaw Reconstruction report". Jigsaw Rule VII.1 (Combinatoric Minimization).
